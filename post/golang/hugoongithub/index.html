Hugo on GitHub

Dec 10, 2018 by

Разнесение кода и статических страниц в двух репозиториях 1. Создаем репозиторий и клонируем его

$ git clone https://github.com/vvuri/hugoblog.git
  1. Ставим hugo
  2. Создаем новый сайт bash $ hugo new site hugoblog
  3. Создаем первый пост bash $ cd vvuri $ hugo new post/start-new-blog.md

  4. Правим контекст в /content/post/*.md

  5. Добаляем .gitignore

  6. Добавляем тему оформления

    $ cd themes 
$ git clone https://github.com/azmelanar/hugo-theme-pixyll.git

    правим файл config.toml

  7. Смотрим локально, что получилось localhosyt:8000

    $ hugo server --buildDrafts --port=8000

    после запуска чистим public

    $ rm -rf public

  8. Заливаем на Git

    $ git add .
$ git commit -m "Initial commit"
$ git push -u origin master

  9. Создаем отдельную ветку внутри основного репозитория

    $ git submodule add -b master https://github.com/vvuri/vvuri.github.io.git public

  10. Создаем статические страницы из всего набора

    $ hugo

    в итоге весь статический сайт в public меняем в config.toml http на https

    baseURL = "https://vvuri.github.io"

  11. Заливаем все на vvuri.github.io

    $ cd public
$ git add .
$ git commit -m "New Public"
$ git push https://github.com/vvuri/vvuri.github.io.git

  12. Если надо удалить субмодуль

    git submodule deinit <path_to_submodule>
git rm <path_to_submodule>
git commit-m "Removed submodule "
rm -rf .git/modules/<path_to_submodule>

  13. В дальнейшем

    $ hugo
$ cd public
$ git add .
$ git commit -m "New Public"
$ git push -u origin master
