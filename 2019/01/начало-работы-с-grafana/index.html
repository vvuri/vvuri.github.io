Grafana - это платформа для визуализации, мониторинга и анализа данных. - основной сайт - Download

  • Debian install $ wget https://s3-us-west-2.amazonaws.com/grafana-releases/release/grafana_4.6.2_amd64.deb $ sudo dpkg -i grafana_4.6.2_amd64.deb
    • Standalone Windows $ wget https://s3-us-west-2.amazonaws.com/grafana-releases/release/grafana-4.6.2.windows-x64.zip $ unzip grafana-4.6.2.windows-x64.zip
